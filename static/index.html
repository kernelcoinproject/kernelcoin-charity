How to Use Your Kernelcoin

Offline secret trading
You can trade the physical secrets offline to control wallets. Keep the secret material safe and treat it like cash or a private key.
Desktop wallet apps
Download our wallet apps for Linux, Windows, and OS X from the Kernelcoin releases page.
Import an existing wallet
You can import a wallet with: kernelcoin-cli importprivkey WIF
Web wallet
We also provide a web wallet you can use from a browser at kernelcoin-webwallet.
Mining
Mine on pool.kernelcoin.org with cpuminer-opt by JayDDee, following the mining guide. Connect to stratum+tcp://pool.kernelcoin.org:3333. New coins become spendable after they mature, which takes about 4 hours.
